# Tool Routing

## Purpose

Tool routing prevents tool sprawl.

## Default Routes

| Need | Route |
| --- | --- |
| Web UI check | Browser |
| Report or SOP | Documents |
| Deck | Presentations |
| Data table | Spreadsheets |
| Motion or video | Hyperframes |

## Rule

Before work starts, decide:

- final artifact
- editable source
- primary tool
- verification method
- manual boundary
