# Codex Setup

## Summary

A useful Codex setup starts with a clear `AGENTS.md`, not with a large plugin list.

## Core Stack

- Browser
- Documents
- Presentations
- Spreadsheets
- Hyperframes

## Rules

- Keep plugin count low.
- Route each task to one primary tool.
- Verify outputs before considering work complete.
- Keep source files editable.
