# Wiki Layer

## Summary

The Wiki Layer turns raw files into structured Markdown notes so the agent does not repeatedly reprocess messy sources.

## Structure

```text
raw/ -> wiki/ -> agent work
instructions/ -> rules for transformation
```

## Benefits

- Lower repeated token cost
- Better continuity
- Easier review
- Local-first knowledge
- Clear links between topics
